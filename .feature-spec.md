## 数据中心 · Data Center

### 数据总览 · Overview

品牌在生成式 AI 搜索中的综合可见性看板，9 段聚合：4 项 KPI + 提及次数趋势 + Top1/Top3 提及率 + 提及率 + 平均位次 + Top 引用 + 文章排名 + 最近会话。

**字段说明**

| 中文名 | 英文名 | 类型 | 说明 |
| --- | --- | --- | --- |
| 提及次数 | mentionCount | int | 目标产品在 AI 回答中被命名提及的总次数 |
| 提及率 | mentionRate | float % | 含品牌提及的对话占总对话数的比例 |
| Top1 提及率 | top1Rate | float % | 品牌列举位次为 1 的对话占比 |
| Top3 提及率 | top3Rate | float % | 品牌列举位次 ≤ 3 的对话占比 |
| 平均提及位次 | avgRank | float | 品牌的平均列举位次（越小越好） |
| Top 引用来源 | topCitation | string(url) | 被模型答案引用最频繁的链接来源 |
| 引用次数 | citationCount | int | 该来源被引用的总次数 |
| 文章标题 | articleTitle | string | 被引用的源文章标题 |
| 会话标题 | conversationTitle | string | AI 平台上的真实对话问题 |
| 平台 | platform | enum | deepseek / wenxin / tongyi / kimi / doubao / chatgpt |
| 场景词 | scenario | string | 监测的业务场景词 |
| 日期区间 | dateRange | object | `{ preset, start, end }`，预设 7d/30d/yesterday 或自定义 |

**接口**

| 接口名 | Interface | 方法 | 参数 | 说明 |
| --- | --- | --- | --- | --- |
| 总览查询 | overview.query | GET /api/data/overview | productId, dateRange, platforms[], scenarios[] | 返回 9 段聚合数据 |
| 导出报告 | overview.export | POST /api/data/overview/export | productId, dateRange, platforms[], scenarios[], format(pdf\|xlsx) | 异步生成下载链接 |

### 场景词数据 · Scenarios

按场景词维度拆解品牌可见性，6 段：词条表 + 提及率 + Top1 提及率 + 平均位次 + 产品对比 + 最近会话。

**字段说明**

| 中文名 | 英文名 | 类型 | 说明 |
| --- | --- | --- | --- |
| 场景词 | keyword | string | 监测的场景词（例："电动车品牌排行榜"） |
| 词条提及率 | rate | float % | 该词条对话中目标产品的提及率 |
| 平均位次 | avgRank | float | 该词条中目标产品的平均排名 |
| 监测平台 | platforms | string[] | 该词条覆盖的 AI 平台代号列表 |
| 会话数 | conversationCount | int | 该词条采集到的对话总数 |
| 对比范围 | scope | enum | own（自家产品）/ all（含竞品） |
| Top N | topN | int | 产品对比柱图取值（5 或 10） |

**接口**

| 接口名 | Interface | 方法 | 参数 | 说明 |
| --- | --- | --- | --- | --- |
| 词条列表 | scenarios.list | GET /api/data/scenarios | dateRange, platforms[], search | 分页词条表格 |
| 提及率趋势 | scenarios.mentionRate | GET /api/data/scenarios/mention-rate | keywords[], dateRange, platforms[] | 趋势 + 平台对比 |
| Top1/Top3 趋势 | scenarios.topRank | GET /api/data/scenarios/top-rank | keywords[], dateRange, platforms[], rank(1\|3) | 趋势 + 平台对比 |
| 平均位次趋势 | scenarios.avgRank | GET /api/data/scenarios/avg-rank | keywords[], dateRange, platforms[] | 趋势 + 平台对比 |
| 产品对比 | scenarios.compare | GET /api/data/scenarios/compare | keyword, scope, topN | 柱图数据 |
| 最近会话 | scenarios.conversations | GET /api/data/scenarios/conversations | keywords[], dateRange, platforms[] | 会话卡片网格 |

### 品牌诊断 · Diagnosis

按需对任意品牌进行多平台多模态扫描，结果落盘为历史记录。

**字段说明**

| 中文名 | 英文名 | 类型 | 说明 |
| --- | --- | --- | --- |
| 诊断记录 ID | diagnosisId | string | 单次诊断的唯一标识 |
| 品牌名 | brand | string | 诊断对象品牌名称 |
| 平台勾选 | platforms | object | `{ web, webDeep, mobile, mobileDeep }` 6 平台多选 |
| 深度思考 | deepThinking | bool | 是否启用模型深度推理 |
| 诊断状态 | status | enum | searching / done / failed |
| 品牌得分 | score | float (0-100) | 综合 5 项指标的总分 |
| 品牌提及率 | rate | float % | 品牌被命名提及的覆盖率 |
| 平均位次 | rank | float | 多品牌列举中的平均位次 |
| 提及次数 | count | int | 命名提及的总次数 |
| 正面情感倾向 | sentiment | float % | 正面 + 中性占比 |
| 诊断时间 | createdAt | datetime | 诊断执行时刻 |

**接口**

| 接口名 | Interface | 方法 | 参数 | 说明 |
| --- | --- | --- | --- | --- |
| 新建诊断 | diagnosis.create | POST /api/data/diagnosis | brand, platforms, deepThinking | 入队执行，返回 diagnosisId |
| 记录列表 | diagnosis.list | GET /api/data/diagnosis | search, dateRange, page, pageSize | 历史诊断记录 |
| 查看详情 | diagnosis.get | GET /api/data/diagnosis/:id |  | 单次诊断完整数据 |
| 重新执行 | diagnosis.retry | POST /api/data/diagnosis/:id/retry |  | 用原参数重跑 |
| 删除记录 | diagnosis.delete | DELETE /api/data/diagnosis/:id |  | 删除单条记录 |

## 知识库 · Knowledge

### 公司管理 · Company

知识库根节点，承载企业主体信息，是产品/人物/媒体等下游资源的归属容器。

**字段说明**

| 中文名 | 英文名 | 类型 | 说明 |
| --- | --- | --- | --- |
| 公司 ID | id | string | 唯一标识 |
| 公司名 | name | string | 工商注册全称 |
| 品牌名 | brandName | string | 品牌方常用简称 |
| 官网 | website | string(url) | 官网链接 |
| 所属行业 | industry | enum | ai-app / saas / hardware / auto / fmcg / enterprise / local / other |
| 一句话定位 | positioning | string | 业务简述 |
| 主营业务 | businessScope | text | 详细业务描述 |
| 品牌别名 | aliases | string[] | 多种名称写法 |
| 关联产品数 | productCount | int | 该公司下产品数（衍生） |
| 关联人物数 | peopleCount | int | 衍生统计 |
| 关联媒体数 | mediaCount | int | 衍生统计 |
| 更新时间 | updatedAt | date | 最后修改时间 |

**接口**

| 接口名 | Interface | 方法 | 参数 | 说明 |
| --- | --- | --- | --- | --- |
| 公司列表 | company.list | GET /api/knowledge/companies | search, page, pageSize | 分页列表 |
| 公司详情 | company.get | GET /api/knowledge/companies/:id |  | 只读详情 |
| 添加公司 | company.create | POST /api/knowledge/companies | name, brandName, website, industry, positioning, businessScope, aliases | 新增公司 |
| 编辑公司 | company.update | PATCH /api/knowledge/companies/:id | 同上字段（任意子集） | 更新 |
| 删除公司 | company.delete | DELETE /api/knowledge/companies/:id |  | 删除（连带处理下游资源） |

### 产品管理 · Product

RAG 知识库根，通过内容块存储品牌/产品的结构化知识；支持文本/文件导入、4 步向导上传与块级编辑。

**字段说明**

| 中文名 | 英文名 | 类型 | 说明 |
| --- | --- | --- | --- |
| 产品 ID | id | string | 唯一标识 |
| 产品名 | name | string | 产品官方名称 |
| 所属公司 ID | companyId | string | 外键 |
| 所属公司名 | companyName | string | 冗余字段，便于列表展示 |
| 内容块 | chunks | Chunk[] | 结构化知识块数组 |
| 更新时间 | updatedAt | date | 最后修改时间 |

**Chunk 对象**

| 中文名 | 英文名 | 类型 | 说明 |
| --- | --- | --- | --- |
| 块 ID | id | string | 唯一标识 |
| 块文本 | text | string (≤240 字符) | 知识内容 |
| 来源类型 | source | enum | manual（手动）/ file（文件） |
| 来源文件名 | sourceName | string | 文件名或"手动添加" |
| Token 数 | tokenCount | int | 块消耗的 Token |

**接口**

| 接口名 | Interface | 方法 | 参数 | 说明 |
| --- | --- | --- | --- | --- |
| 产品列表 | product.list | GET /api/knowledge/products | search, companyId, page, pageSize | 列表（支持公司筛选） |
| 产品详情 | product.get | GET /api/knowledge/products/:id |  | 含全部 chunks |
| 添加产品 | product.create | POST /api/knowledge/products | name, companyId, sources[] | sources 含 textInput 或 file 上传项 |
| 文件上传 | product.uploadFile | POST /api/knowledge/products/upload | file (multipart) | 返回解析后的 chunks（用于预览） |
| 编辑产品 | product.update | PATCH /api/knowledge/products/:id | name, companyId, chunks | 整体替换 chunks 或部分增改删 |
| 删除产品 | product.delete | DELETE /api/knowledge/products/:id |  | 连同所有 chunks 删除 |
| 块级去重 | product.dedupeChunks | POST /api/knowledge/products/dedupe | existingChunks, incomingChunks | 返回 fresh / skipped 统计 |

### FAQ 管理 · FAQ

品牌常见问答库，支持公司级和产品级两种归属粒度，单条手填或批量导入。

**字段说明**

| 中文名 | 英文名 | 类型 | 说明 |
| --- | --- | --- | --- |
| FAQ ID | id | string | 唯一标识 |
| 问题 | question | string | 常见问题 |
| 答案 | answer | text | 标准答案 |
| 归属类型 | scope | enum | company（公司级）/ product（产品级） |
| 所属公司 ID | companyId | string | 公司外键（必填） |
| 所属公司名 | companyName | string | 冗余展示字段 |
| 所属产品 ID | productId | string \| null | 产品外键（scope=product 时必填） |
| 所属产品名 | productName | string \| null | 冗余展示字段 |
| 更新时间 | updatedAt | date | 最后修改时间 |

**接口**

| 接口名 | Interface | 方法 | 参数 | 说明 |
| --- | --- | --- | --- | --- |
| FAQ 列表 | faq.list | GET /api/knowledge/faqs | search, scope, companyId, productId, page, pageSize | 多维筛选 |
| FAQ 详情 | faq.get | GET /api/knowledge/faqs/:id |  | 只读详情 |
| 添加单条 | faq.create | POST /api/knowledge/faqs | question, answer, scope, companyId, productId? | 单条手填 |
| 批量导入 | faq.batchImport | POST /api/knowledge/faqs/batch | scope, companyId, productId?, items[]（粘贴文本或文件） | 同归属下按 question 去重后落库 |
| 文本解析 | faq.parseText | POST /api/knowledge/faqs/parse-text | text | 返回解析出的 Q/A 对（不入库） |
| 文件解析 | faq.parseFile | POST /api/knowledge/faqs/parse-file | file (multipart) | 返回解析出的 Q/A 对（不入库） |
| 编辑 FAQ | faq.update | PATCH /api/knowledge/faqs/:id | question, answer, scope, companyId, productId | 单条更新 |
| 删除 FAQ | faq.delete | DELETE /api/knowledge/faqs/:id |  | 单条删除 |
